<h2>Features</h2>

Here're some of the project's best features:

*   Minimal UI
*   Repeat Shuffle mode
*   Background playback
*   Media buttons support (Headphones or car buttons)
*   Wear OS support (Planned)
*   Lyric support (Planned)

  
  
<h2>Built with</h2>

Technologies used in the project:

*   Jetpack Compose
*   Jetpack Media3-ExoPlayer
*   Jetpack Navigation
*   Hilt
